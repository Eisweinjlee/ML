2.2 Function-space View

We use a Gaussian process (GP) to describe a distribution of functions.

Definition 2.1 A Gaussian process *is a collection of random variables, any finite number of which have a joint Gaussian distribution.

We define mean function m(x)m(\mathbf{x})m(x) and the covariance function k(x,x′)k(\mathbf{x},\mathbf{x}&#x27;)k(x,x′) of a real process f(x)f(\mathbf{x})f(x) as m(x)=E[f(x)],m(\mathbf{x})=\mathbb{E}[f(\mathbf{x})],m(x)=E[f(x)], k(x,x′)=[(f(x)−m(x))(f(x′)−m(x′))],k(\mathbf{x},\mathbf{x}&#x27;)=\mathbb[(f(\mathbf{x})-m(\mathbf{x}))(f(\mathbf{x}&#x27;)-m(\mathbf{x}&#x27;))],k(x,x′)=[(f(x)−m(x))(f(x′)−m(x′))], and write the Gaussian process as f(x)∼GP(m(x),k(x,x′))f(\mathbf{x})\sim\mathcal{GP}(m(\mathbf{x}),k(\mathbf{x},\mathbf{x}&#x27;))f(x)∼GP(m(x),k(x,x′))

In stochastic process, the index set X\mathcal{X}X of random variables is time ttt. However, in Gaussian process the set X\mathcal{X}X is more general e.g. RD\mathbb{R}^DRD.

A simple example can be obtained from a Bayesian linear regression model f(x)=ϕ(x)Twf(\mathbf{x})=\mathbf{\phi}(\mathbf{x})^T\mathbf{w}f(x)=ϕ(x)Tw with the prior w∼N(0,Σp)\mathbf{w}\sim\mathcal{N}(\mathbf{0},\Sigma_p)w∼N(0,Σp​). Therefore, f(x)f(\mathbf{x})f(x) is jointly Gaussian with the mean and the covariance as E[f(x)]=ϕ(x)TE[w]=0,\mathbb{E}[f(\mathbf{x})]=\phi(\mathbf{x})^T\mathbb{E}[\mathbf{w}]=0,E[f(x)]=ϕ(x)TE[w]=0, E[f(x)f(x′)]=ϕ(x)TE[wwT]ϕ(x′)=ϕ(x)TΣpϕ(x′) \mathbb{E}[f(\mathbf{x})f(\mathbf{x&#x27;})]=\phi(\mathbf{x})^T\mathbb{E}[\mathbf{w}\mathbf{w}^T]\phi(\mathbf{x&#x27;})=\phi(\mathbf{x})^T\Sigma_p \phi(\mathbf{x&#x27;})E[f(x)f(x′)]=ϕ(x)TE[wwT]ϕ(x′)=ϕ(x)TΣp​ϕ(x′)

In Bayes linear regression w\mathbf{w}w is a parameter vector so that the regression has only one basis function. However, in Gaussian process regression it is a random varible, so it can be seen as having an infinite number of basis function.

We will study different covariance functions, which imply the corelation between two inputs. In this chapter we would run on squared exponenetial (SE) covariance function: k(x,x′)=cov(f(x),f(x′))=exp⁡(−12∣x−x′∣2)k(\mathbf{x},\mathbf{x}&#x27;)=\text{cov}(f(\mathbf{x}),f(\mathbf{x}&#x27;))=\exp(-\frac{1}{2}|\mathbf{x}-\mathbf{x}&#x27;|^2)k(x,x′)=cov(f(x),f(x′))=exp(−21​∣x−x′∣2) We also want to introduce the distance effect, then introduce a positive constant lll by replacing ∣x−x′∣|\mathbf{x}-\mathbf{x}&#x27;|∣x−x′∣ with ∣x−x′∣/l|\mathbf{x}-\mathbf{x}&#x27;|/l∣x−x′∣/l.
