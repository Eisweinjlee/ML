2.3 Briefly about the Hyperparameters

Remember if we construct the covariance function from a SE covariance function in this form ky(xp,xq)=σf2exp⁡(−12l2(xp−xq)2)+σn2δpqk_y(x_p,x_q)=\sigma_f^2\exp(-\frac{1}{2l^2}(x_p-x_q)^2)+\sigma_n^2\delta_{pq}ky​(xp​,xq​)=σf2​exp(−2l21​(xp​−xq​)2)+σn2​δpq​ where δpq\delta_{pq}δpq​ is the kronecker delta. Then we have three parameters, the length-scale lll, the signal variance σf2\sigma_f^2σf2​ and the noise variance σn2\sigma_n^2σn2​.

Length-scale lll
Sometimes we change the length-scale and use marginal likelihood to optimize other two parameters. We can see:

  • If the lll is too short, the regression might be over-fitting.
  • If the lll is too long, the regression might be under-fitting.

2.4 Decision Theory on Regression

In practical a-pplications, we need to know the the deviation between our prediction yguessy_{\text{guess}}yguess​ and the ture value yturey_{\text{ture}}yture​. Then we define a loss function L(yguess,yture)\mathcal{L}(y_{\text{guess}},y_{\text{ture}})L(yguess​,yture​) for the optimization purpose.

In non-Bayesian paradigms, the model is trained to minimize the empirical risk (or loss function).
In contrast, in the Bayesian setting there is a clear seperation between likelihood nad the loss function. The likelihood function is telling the deviation between nosiy measurements and the underlying noisy-free model. The loss function is the consequence of making choice or taking act with purposed strategy.

However, we don’t know the ytruey_{\text{true}}ytrue​. Instead, we minimize the expected loss or risk by averaging our model’s opinion. We can select a suitable loss function L\mathcal{L}L for different tasks.
In general, the value yguessy_{\text{guess}}yguess​ that minimized the risk for L=∣yguess−y∗∣\mathcal{L}=|y_{\text{guess}}-y_*|L=∣yguess​−y∗​∣ is the median of p(y∗∣x∗,D)p(y_*|\mathbf{x}_*,D)p(y∗​∣x∗​,D), while for the squared loss L=(yguess−y∗)2\mathcal{L}=(y_{\text{guess}}-y_*)^2L=(yguess​−y∗​)2 it is the mean of the distribution.

2.6 Smoothing, Weight Functions and Equivalent Kernels

Linear smoother
Gaussian process aims to remove the noise ε\varepsilonε of the observation fff, by computing the noisy observations with fˉ(x∗)=k(x∗)T(K+σn2I)−1y\bar f(\mathbf{x}_*)=\mathbf{k}(\mathbf{x}_*)^T(K+\sigma_n^2 I)^{-1}\mathbf{y}fˉ​(x∗​)=k(x∗​)T(K+σn2​I)−1y. We can prove the smoothing (see the book), because smoother eliminates the component in y\mathbf{y}y along eigenvector ui\mathbf{u}_iui​ of K=Σi=1NγiuiuiTK=\Sigma^N_{i=1}\gamma_i\mathbf{u}_i\mathbf{u}_i^TK=Σi=1N​γi​ui​uiT​.

Weight function
We define a vectors of functions h(x∗)=(K+σn2I)−1k(x∗)\mathbf{h}(\mathbf{x}_*)=(K+\sigma_n^2 I)^{-1}\mathbf{k}(\mathbf{x}_*)h(x∗​)=(K+σn2​I)−1k(x∗​), thus we have fˉ(x∗)=h(x∗)Ty\bar f(\mathbf{x}_*)=\mathbf{h}(\mathbf{x}_*)^T\mathbf{y}fˉ​(x∗​)=h(x∗​)Ty. It can be seen that for a fixed test point x∗\mathbf{x}_*x∗​, the h(x∗)\mathbf{h}(\mathbf{x}_*)h(x∗​) gives the vector of weights applied to targets y\mathbf{y}y, therefore h(x∗)\mathbf{h}(\mathbf{x}_*)h(x∗​) is called weight function.

Note the difference between a linear model and a linear smoother. The prediction of a linear model is a linear combination of the inputs x\mathbf{x}x, while the prediction of a linear model is a linear combination of the training set targets y\mathbf{y}y.
